# OfficeFlowAI v0

เว็บแอปตัวอย่างสำหรับงานขายและเอกสารออฟฟิศ สร้างด้วย HTML + CSS สำหรับ Frontend และ JavaScript บน Node.js สำหรับ Backend โดยยังไม่พึ่งพาแพ็กเกจภายนอก

## ฟีเจอร์หลัก

- ระบบล็อกอินแบบ Demo (`admin` / `admin123`)
- หน้า Dashboard หลังล็อกอิน แสดงยอดรวม Quotation, PO, ยอดชำระแล้ว และ PO ที่รอชำระ
- สร้างใบ Quotation จากรายการสินค้า/บริการ จำนวน และราคา พร้อมคำนวณ VAT 7%
- รับข้อมูลใบ PO จากข้อความ OCR และบันทึกลงฐานข้อมูล JSON
- ยืนยันการชำระเงินของ PO เพื่อสร้างใบเสร็จ PDF อัตโนมัติ

## วิธีรัน

```bash
npm start
```

จากนั้นเปิดเว็บที่ <http://localhost:3000>

## ตรวจสอบ Syntax

```bash
npm run check
```

## โครงสร้างข้อมูล

- `data/db.json` ถูกสร้างอัตโนมัติเมื่อรันระบบ และถูก ignore จาก git
- `receipts/*.pdf` ถูกสร้างอัตโนมัติเมื่อยืนยันการชำระเงิน และถูก ignore จาก git
